\section{Introduction}\label{introduction}

Managing 300+ Git repositories across 20GB of research data requires a
robust, automated backup strategy. This guide walks through setting up a
three-tier backup system that provides Git-level versioning, real-time
cloud sync, and comprehensive system backups.

\subsection{Backup Strategy Overview}\label{backup-strategy-overview}

Our approach uses three complementary layers:

\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\tightlist
\item
  \textbf{Automated Git commits and pushes} (every 15 minutes)
\item
  \textbf{Cloud synchronization} (real-time via Google Drive/Dropbox)
\item
  \textbf{Time Machine backups} (hourly system-wide backups)
\end{enumerate}

This ensures your research is protected against hardware failure,
accidental deletion, Git corruption, and provides easy access across
devices.

\section{Setting Up Time Machine}\label{setting-up-time-machine}

Time Machine provides system-wide backup protection and serves as your
safety net for everything beyond Git repositories.

\subsection{Initial Time Machine
Setup}\label{initial-time-machine-setup}

\subsubsection{Step 1: Connect Your USB
Drive}\label{step-1-connect-your-usb-drive}

\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\tightlist
\item
  Connect your 1TB USB drive to your MacBook
\item
  When prompted, \textbf{do not} use it for Time Machine yet - we'll
  configure this properly first
\end{enumerate}

\subsubsection{Step 2: Format the Drive (if
needed)}\label{step-2-format-the-drive-if-needed}

\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\tightlist
\item
  Open \textbf{Disk Utility} (Applications \textgreater{} Utilities
  \textgreater{} Disk Utility)
\item
  Select your USB drive from the sidebar
\item
  Click \textbf{Erase}
\item
  Choose format: \textbf{Mac OS Extended (Journaled)} or \textbf{APFS}
  (recommended for newer Macs)
\item
  Name it something like ``Research Backup''
\item
  Click \textbf{Erase}
\end{enumerate}

\subsubsection{Step 3: Configure Time
Machine}\label{step-3-configure-time-machine}

\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\tightlist
\item
  Open \textbf{System Preferences} \textgreater{} \textbf{Time Machine}
\item
  Click \textbf{Select Backup Disk}
\item
  Choose your USB drive
\item
  Click \textbf{Use Disk}
\item
  If prompted about encryption, choose \textbf{Encrypt Backup} for
  security
\end{enumerate}

\subsubsection{Step 4: Customize Time Machine
Settings}\label{step-4-customize-time-machine-settings}

\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\tightlist
\item
  Click \textbf{Options} in Time Machine preferences
\item
  Add any folders you want to exclude (like Downloads, Trash, etc.)
\item
  \textbf{Important}: Do NOT exclude \texttt{\textasciitilde{}/prj} - we
  want this backed up
\item
  Ensure ``Back up while on battery power'' is enabled if desired
\end{enumerate}

Time Machine will now automatically backup your entire system (including
\texttt{\textasciitilde{}/prj}) every hour when the USB drive is
connected.
